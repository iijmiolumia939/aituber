# Popica Phase 1 QC Gates

## Gate運用ルール

- 各バッチ完了時に対応するGateをチェックする。
- Gateが未達なら次バッチへ進まない。
- 破綻を発見したら該当レイヤーを `InProgress` に戻して修正する。

## G1: Face / Eye / Mouth

- [ ] 顔ベースと首の境界に欠けがない
- [ ] 瞬き0%/50%/100%で白目はみ出しがない
- [ ] 虹彩が左右で高さズレしない
- [ ] 口開閉0%/50%/100%で口内が破綻しない

## G2: Hair

- [ ] 前髪を上げても額の穴が出ない
- [ ] 横髪と頬の重なり順が自然
- [ ] 後ろ髪のエッジに透けや欠けがない
- [ ] ツインテ根元の接続に不自然な段差がない

## G3: Body / Arm / Skirt

- [ ] 首と胴体の接続に隙間がない
- [ ] 腕回転時に袖境界の破綻が目立たない
- [ ] スカート上端と腰の接続に穴がない
- [ ] 左右腕の太さ・長さの視覚バランスが一致

## G4: Ribbon / Final Pass

- [ ] 頭リボンが髪の前後関係で破綻しない
- [ ] 胸・腰リボンの重なり順が自然
- [ ] すべての必須レイヤーが `Done`
- [ ] Cubism最小パラメータで重大破綻がない

## フリーズ条件（Phase 1完了判定）

- [ ] `phase1_layers.csv` の status が全件 `Done`
- [ ] `phase1_parameters.json` の全パラメータを実装済み
- [ ] 30分相当の確認で目立つ破綻が再現しない
