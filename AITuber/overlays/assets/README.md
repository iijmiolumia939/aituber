# オーバーレイ画像アセット

`tha_broadcast.html` が使用する画像パーツ一覧です。  
リファレンス画像から以下のパーツを切り出して、このディレクトリに配置してください。

## 必須アセット

| ファイル名 | 内容 | 推奨サイズ | 備考 |
|---|---|---|---|
| `bg.png` | 背景（ボケ＋ハート＋キラキラ） | 1920×1080 | フルスクリーン背景。アバターやUIパネルは含めない |
| `frame_chat.png` | チャットパネル枠 | 580×780 | ピンク枠 + 白半透明内部。「チャット」ヘッダ含む。パネル内側は透明 |
| `frame_menu.png` | メニューパネル枠 | 300×450 | 「きょうのメニュー」ヘッダ含む。内側は透明 |
| `nameplate.png` | ネームプレート | 460×220 | 「天導・ルミナ」+ 「ポップ種族AITuber」のデコレーション全体 |

## 任意アセット

| ファイル名 | 内容 | 推奨サイズ | 備考 |
|---|---|---|---|
| `frame_subtitle.png` | 字幕バー枠 | 760×100 | 字幕テキストの背景フレーム。無い場合はCSS代替 |
| `frame_info.png` | 情報パネル枠 | 380×170 | 下部の情報パネル（2つで共通利用可）。無い場合はCSS代替 |
| `live_badge.png` | LIVE バッジ | 120×44 | LIVE 表示バッジ。無い場合はCSS代替 |

> **パーティクル (ハート・キラキラ)** は tsParticles (MIT) で JS 実装済み。画像アセット不要。
> CDN 未到達時は絵文字フォールバックが動作。

## 切り出し手順

1. リファレンス画像を Photoshop / GIMP / Figma 等で開く
2. 各パネル枠を選択し、**透過PNG** でエクスポート
   - パネル枠のみ切り出し（中のテキスト内容は不要、フレーム装飾だけ）
   - ネームプレートはテキスト込みでOK
3. 背景は **アバター・UIパネルを除外** して切り出す
   - コンテンツ認識塗りつぶし等で埋めるか、元の背景パターンだけ残す
4. このディレクトリ (`overlays/assets/`) に配置

## 注意事項

- すべて **透過PNG** (RGBA) で保存
- ファイルサイズ目安: 各パーツ 100KB〜500KB（PNG圧縮推奨）
- `bg.png` は 1MB 以内推奨
- アセットが無い場合でも `tha_broadcast.html` はCSS代替で動作します

## JS ライブラリ (CDN)

| ライブラリ | ライセンス | 用途 | CDN |
|---|---|---|---|
| `tsparticles-slim` v2 | MIT | ハート・キラキラ・紙吹雪パーティクル | jsdelivr |
| `obs-websocket-js` v5 | MIT | OBS シーン自動切替 | jsdelivr |

両ライブラリは CDN から defer ロード。読み込めない場合は自動的に無効化される（フォールバック動作）。

### OBS WebSocket 設定

OBS Studio 28+ に同梱の obs-websocket (ポート 4455) と連携。
Orchestrator から `{ type: "scene", scene: "opening" }` 等の WS メッセージ受信時に
自動で OBS シーンを切り替える。

シーンマッピング:
- `opening` → OBS `Opening`
- `chat` → OBS `Main`
- `game` → OBS `Game`
- `ending` → OBS `Ending`

OBS が未起動 or パスワード不一致でも overlay は正常動作する。
