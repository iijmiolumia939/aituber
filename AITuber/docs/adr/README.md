# Architecture Decision Records

This directory stores immutable Architecture Decision Records for the AITuber repository.

Rules:
- Add new decisions as new files. Do not rewrite accepted history.
- Mark replaced decisions as `Superseded` and point to the replacement ADR.
- Keep implementation details in code, tests, and executable tooling.
- Use ADRs for stable decisions that GitHub Copilot and humans both need to rediscover safely.

Format:
- `0001-short-title.md`
- `Status`, `Date`, `Context`, `Decision`, `Consequences`, `References`
