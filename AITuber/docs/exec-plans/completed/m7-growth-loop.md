### `orchestrator/growth_loop.py` (新規)

#### `GrowthLoopResult` dataclass

```python
@dataclass
class GrowthLoopResult:
    n_generated: int   # valid proposals generated by LLM
    n_approved: int    # proposals approved and written to policy
    n_rejected: int    # proposals rejected (or 0 if n_generated==0)
```

#### `GrowthLoop` クラス

```python
class GrowthLoop:
    def __init__(
        self,

        auto_approve: bool = False,
        auto_reject: bool = False,

    async def run(self) -> GrowthLoopResult: ...
```

#### パイプライン実装

```python
async def run(self) -> GrowthLoopResult:
    # Step 1: ReflectionCLI -> staging file
    cli = ReflectionCLI(
        gaps_dir=self.gaps_dir,
        policy_path=self.policy_path,
        top_n=self.top_n,
        dry_run=False,
        backend=self._backend,
        output_path=self.staging_path,
    )
    await cli.run()

    # Step 2: count what was staged
    n_generated = _count_staging(self.staging_path)
    if n_generated == 0:
        return GrowthLoopResult(0, 0, 0)

    # Step 3: ApproveCLI -> policy update
    approve = ApproveCLI(
        staging_path=self.staging_path,
        policy_path=self.policy_path,
        auto_approve=self.auto_approve,
        auto_reject=self.auto_reject,
        input_fn=self.input_fn,
    )
    approve.run()

    # Step 4: count results
    n_approved = _count_approved(policy_before, policy_after)
    return GrowthLoopResult(n_generated, n_approved, n_generated - n_approved)
```
